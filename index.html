Mission Statement:

The goal of the Alysha project is to build an A.I. robot from scratch using materials and components readily available to a home hobbyist. The project will use Arduinos, STM-32s, Raspberry Pi, and a Jetson Nano. The body will consist of a base on wheels, a couple arms, and eventually a human-like head shown in the photo at the top of the page. The robot will be able to carry on meaningful, albeit low-level conversations. This will be accomplished using the Microsoft Speech Platform and their Grammar Builder. There are six major layers of this robot project. They are listed under the Project Overview above. Programming languages used will be C#, C++, and Python.
